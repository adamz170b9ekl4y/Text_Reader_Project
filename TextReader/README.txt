#TextReader

TextReader is a simple JavaFx application which can read and display text
files into a ebook like format. Basic comforts are include such as search,
and mode (dark and light). Other functions will be included as time goes on.  

#Usage
All options can be preformed using the top menu bar, in a similar to MS notepad
and other text-reading applications.
